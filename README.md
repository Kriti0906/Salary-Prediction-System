# Salary Prediction System

A Machine Learning project that predicts employee salaries based on **Years of Experience** and **Job Rate** using a **Linear Regression** model. The project includes data analysis, visualization, model training, evaluation, and a Streamlit-based user interface for salary prediction.

## Features

* Employee salary prediction
* Data analysis and visualization
* Linear Regression model training
* Model evaluation
* Interactive Streamlit interface

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Joblib
* Streamlit

## Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Model Training using Linear Regression
5. Model Evaluation
6. Model Saving with Joblib
7. Salary Prediction through Streamlit

```bash
pip install -r requirements.txt
streamlit run app.py
```
